# Query builder

The purpose of this project is to define graph filters and traversals using yaml/json.


## Features 

- [ ] Filter Nodes
- [ ] Filter Edges 
- [ ] Filter by Properties 
- [ ] Filter by Edges Count
- [ ] order by
- [ ] Travese via edges 
- [ ] Traverse to Nodes 
- [ ] Subgraph
